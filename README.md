## 📌 Мои проекты

### 🔍 [JSON Validator & Beautifier Bot](https://github.com/VanishID/tg_TestBuddyBot)
Telegram-бот для проверки и форматирования JSON. Присылает красивый JSON или указывает на ошибку. Помощник для QA при разборе логов и ответов API.

### 🧪 [User Generator Bot](https://github.com/VanishID/tg_UserGeneratorBot)
Генератор случайных тестовых пользователей: имя, email, адрес, логин, пароль. Нужное количество — одной командой. Ускоряет подготовку тестовых сред.

### ✅ [Автотесты API (Python + Pytest + Requests)](https://github.com/VanishID/Python-Pytest-Requests)
Автоматизированные проверки REST API: статус-коды, структура ответов, валидация данных тренеров. Написаны с использованием pytest и requests.

### 🌐 [Сайт-визитка QA-инженера](https://github.com/VanishID/VanishID.github.io)
Персональный лендинг с контактами, увлечениями и ссылками. Чистая вёрстка на HTML и CSS.

### 🌲 [Cypress UI Тесты](https://github.com/VanishID/Cypress)
Автотесты пользовательского интерфейса на JavaScript + Cypress.  
Проверка сценариев авторизации, валидации @ в логине, восстановления пароля и приведения логина к строчным буквам на сайтах `login.qa.studio` и `pokemonbattle.ru`.
